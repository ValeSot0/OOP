Definicion de OOP

La POO es un paradigma de programación que organiza el código en objetos. Estos objetos representan entidades del mundo real, como personas, coches, cuentas bancarias, etc. Cada objeto tiene sus propias propiedades (atributos) y comportamientos (métodos).

Ejemplo:

  • Objeto: Un coche.
  • Propiedades: Color, marca, modelo, número de puertas.
  • Comportamientos: Encender, acelerar, frenar, girar.

Herencia

Es una relación jerárquica entre clases, donde una clase (hija o subclase) hereda las propiedades y métodos de otra clase (padre o superclase).

Polimorfismo

Es la capacidad de que objetos de diferentes clases respondan de manera diferente al mismo mensaje.

Abstracción

Es el proceso de identificar las características esenciales de un objeto y omitir los detalles innecesarios.

Encapsulamiento

Es el mecanismo que oculta los detalles internos de un objeto, exponiendo solo una interfaz para interactuar con él.
